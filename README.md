Indo MOS (Indonesia Macet of Shame) - BETA
==========================================

Aplikasi simpel yang mengcapture kemacetan lalu lintas dengan menggunakan Google Map.

# Fitur

Untuk saat ini Indo MOS masih dalam tahap brainstorming dan pengembangan awal. Akan tetapi akan diusahakan untuk membuat Web UI yang dapat digunakan untuk melihat komparasi lalu lintas dari waktu ke waktu.
